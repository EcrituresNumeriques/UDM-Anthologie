# UDM-Anthologie

The project contain Front Backend and REST Api

## Requirements:
[Docker] and Git have to be installed on your server.

## Installation:
```sh
$ git clone http://project
$ docker-compose up
```

## Technologies :

##### Front:
* [VueJs] - js Framework
* [BootStrap] - css Framework

##### Back:
* [Symfony] - A strong and powerfull php framework
